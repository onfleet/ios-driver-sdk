AnalyticsController

public final class AnalyticsController

Each analytics period has it’s own AnalyticsPeriodController. These are managed by this controller. When current account is switched, all controllers must be reloaded with data for new org.

  • cacheDirectoryUrl

    Directory url where analytics caches files will be stored; if nil default framework caches will be used. This is useful if you wish to save files into shared folders with your widgets.

    Note

    this property must be set before DriverContext config method is called.
