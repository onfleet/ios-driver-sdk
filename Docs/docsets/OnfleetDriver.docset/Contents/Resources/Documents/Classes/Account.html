Account

@objcMembers
public class Account : NSObject, Codable

A model representing an organization account in SessionManaging.

Account can have multiple states. If driver is invited into an org, account is in INVITED state and isPending returns true. Driver can either accept or reject invitation. If account is accepted this account is set to ACCEPTED and isAccepted returns true; REJECTED and isRejected respectively. Pending or rejected accounts can’t go on duty, start tasks, etc. Refer to SessionManaging for more info.
